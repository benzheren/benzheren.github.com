--- layout: post title: Install MySQLdb Python driver on Mac OS ---

If you get this error when you try to build it, please read the README carefully because you need to install setuptools first.

ImportError: No module named setuptools

If you get this error when you try to build it, it is because mysql is installed in a customized path on your Mac.

EnvironmentError: mysql_config not found

You just need to modify this link in the site.cfg.

#mysql_config = /usr/local/bin/mysql_config
