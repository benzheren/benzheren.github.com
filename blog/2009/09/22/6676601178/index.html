An Annoying IE Bug

Sep 22nd, 2009

Yesterday, I came across a very annoying IE bug: IE XMLHTTP implementation turns a standard 204 response code into a pretty random 1223 status code. Apparently it is an IE bug for a while, at least for both IE 7 and IE 8. And I found out a pretty cool website to track IE bugs.

I am not an anti-Microsoft person, but Microsoft Internet Explorer is really one of a kind. These days when the whole industry is trying to embrace standards, IE products are behind the industry standards movement. IE6 is already a disaster for web developers and a lot of new websites already stop supporting it. IE8 has made some good improvements, hope Microsoft can incorporate more open standards into its IE product and fix those annoying bugs.
