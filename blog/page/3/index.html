IE Conditional Comments With HAML

Aug 17th, 2011

I am using HAML, SASS with Rails for most of the front end code. I was trying to add some IE conditional CSS include in the header and HTML should look like this:
<!–[if (!IE)|(gte IE 8)]><!–>
<link href=”some.css” media=”screen” rel=”stylesheet” type=”text/css” />
<!–<![endif]–>

It is a little bit different from straightforward IE conditional comments like this:
<!–[if (!IE)|(gte IE 8)]>
<link href=”some.css” media=”screen” rel=”stylesheet” type=”text/css” />
<![endif]–>

The way I implementes in HAML is:
/[if (!IE)|(gte IE 8)]
!= “<!–>”
= stylesheet_link_tag ‘some’, :media => ‘screen’
!= “<!–”

I am not sure if there is an easier way to do this in HAML, but my solution works!

工行开户趣闻

Aug 14th, 2011

回国后天天都能遇到有趣的事情：今天去工行开户，因为住的对面就有一个，也比较相信工行是大银行，开开心心去开户了。结果到了那里，搞笑的事情发生了：

第一，根据工行规定，每个身份证号只能有一个网银登陆，因为我原来很久以前用的学校的工行卡开通过网银了（我自己都不记得，多少年前的事情了），所以我要注销原来那个，要过一天才可以开通新的。

第二，更搞笑的在后面，银行的人一查我名字，我好好的一名字，后面尽然加了10个@符号，我是搞技术的，一看就知道肯定是哪个搞数据库或者系统的对于短的名字自定义加了一些字符凑齐比特数，这下银行的人也傻眼了，说没法开户，因为我身份证名字和系统名字差了10个@，我必须要到我开户行去更新我的“真名”。

哎，这么大一个上市的国有银行，当年，甚至现在还有可能是全世界市值最高的银行，技术竟然是如此，我承认银行的技术本来就比较保守落后，可是做成这样地步的，实在是让我很担心啊。在微博上发了一贴，结果我朋友幽默了一把，说估计工行和微博有合作关系，特别喜欢@符号，只能是怒其不争啊。

Hosting Service in China

Aug 13th, 2011

When you start working on an Internet startup, one of few things you will think about first is how I should host my service. In US, where Internet infrastructure is ahead of the rest of the world, this becomes less a problem for startup because there are so many choices like AWS, GAE or even Microsoft Azure. However, situation in China is very different: first, both AWS and GAE do not have servers in mainland China, which means it is really slow to access either of them; second, legally it is not a good idea to use foreign servers anyway, because in order for you to run Internet business in China, you have to register your website with government and that requires you have serves physically located in China.

In the old time, some Chinese startups will start with a foreign VPS (usually in US) such as Linode for the development and initial launch while they wait for the approval of government organizations. After that, the startup has to buy its own servers and pay data centers to put their servers on the rack which is a pretty big investment upfront for a small startup.

With the development of Chinese Internet infrastructure and the boom of tech startups, these days there are more and more local companies start to offer VPS or cloud hosting services. Companies I am aware of are:

  • http://aliyun.com: this is a service provided by Alibaba, one of the 3 biggest Internet companies in China. Their service looks like a VPS service with a pretty clear fee structure. It starts with 199 RMB a month (about 30 USD) for a server. It provides a beta version cloud database service which does backup and a lot of maintenance work for you. It supports only MySQL for now.

  • http://netbank.cn: this is a HangZhou company which provides VPS service as well. It provide three basic plans and on top of that you can customize according to your need. For a Intel XEON520 single core, 2GB memory and 30GB harddrive server with 100MB bandwidth from China Telecom (this is not the amount, just the speed)  , it costs you around 3000RMB/year.

  • http://xingcloud.com: this is a startup from Kai-Fu Lee’s Innovation Works, currently it markets itself as a cloud platform for game developers. With the reputation of Kai-Fu Lee and the strong team behind it, XingCloud could be an interesting player to watch in this area.


A very interesting thing to note about hosting service in China is: you need to pay for bandwidth of different ISPs, because if you don’t do that, your service will be slow for users using that ISP. There are six major ISPs here, and two of them are the biggest: China Telecom and China Net Communication (CNC). For a startup, you probably should pay bandwidth for these 2 ISPs. Because of this bandwidth thing, the cost of Chinese Internet Video companies are much higher than Youtube.

What did we do?

Before we got here, we started with a Linode monthly account. We have been using that for the past week, it is just super slow because of GFW and the physical distance. Sometimes our terminal window just halts there. So we have decided to switch to a local provider since we can try out their service 3 months for free. That will make huge difference because the average ping value for Linode server is about 350ms while the local one is about 5ms. By the way, there is a service in China to test your ping value from different parts of the country with different ISPs. 

Hope this information can give you an idea how hosting works in China. =)

不适应中国生活的中国人

Aug 10th, 2011

回国已经一个星期了，和八月的华东地区天气一样火热的是当今年轻人的创业热情，在我们的cowork space外面的电子广告牌滚动播放着的是“创业大赛”的信息，看着19岁休学创业的同学，和16岁就辍学已经做到第三个公司的朋友，感觉到我已经不是那么年轻了。即使过年去3年，我年年回国，但是国外5年的生活，让我在很多方面已经不适应国内的生活了，就更不要提我的兄弟Matt了，虽然很多人都列举过你会不适应的方面，我在这里就多啰嗦一下，说说我自己的感受：

  1. 过马路：我在国内也不开车，基本和我在纽约和旧金山的时候一样，出门靠腿。在国内行人过街，实在是需要一些胆量和技术的，拐弯的车辆一般基本上对行人是熟视无睹的，疯狂的助动车是雪上加霜。在非路口的斑马线的地方，其实应该设置一下红绿灯，不然的话斑马线基本起不到什么作用的。

  2. 洗手：国内的饭店同时有洗手液和干手机的地方实在是太少了，那里的员工如厕后洗手不？GDP不停地在涨，我们在的cowork space的厕所还是和我大学时候一样的蹲坑式，无厕纸，无洗手液。

  3. 银行：我现在基本上就不用在国内进行网上购物了，大部分银行的安全登陆系统还在用ActiveX的插件，不用Windows的我基本就傻眼了。无业的我估计目前申请信用卡有点难度，没有信用卡连个支付宝都没法好好用，支付宝似乎支持Safari，但是那个安全插件装了半天也没用，实在是很晕啊。

  4. 美式快餐：习惯了国外快餐的自助式服务，你可以自己随便拿纸巾，随便拿番茄酱，在国内还真是不适应，昨天去吃个麦当劳，多要了个番茄酱，还被服务员用非常奇异的眼光看了一眼，感觉是见到外星人似的。

  5. 洗衣服：国内要租个有烘干机的公寓估计价格不菲，现在我也犯不着，所以晾衣服成为了一门必修课。


至于像物价飞涨，GFW威武之类的我就不多提了，洋洋洒洒啰嗦了一下，只是想给大家提个醒，国外待了时间久了，估计得花点时间来慢慢适应国内生活，有机会我可以叫Matt同学补充一下，写个姊妹篇：不适应中国生活的美国人。
